ФОРМАТЫ

Адреса выдачи

Один токен можно использовать в нескольких форматах.

Base64/sub/TOKENОсновной вариант для v2rayN, v2rayNG и совместимых клиентов.
Обычная ссылка/sub/TOKEN?format=plainVLESS-ссылка без кодирования, удобная для проверки.
JSON/sub/TOKEN?format=jsonСтруктурированный ответ для интеграций.
!

Токен является секретом. После его перегенерации прежний адрес сразу перестаёт работать.
