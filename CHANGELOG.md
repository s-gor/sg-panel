# Changelog

## 0.9.5 — VLESS outbound transports

- добавлены кнопки выбора протокола, транспорта и защиты в разделе Outbounds;
- поддержаны VLESS + RAW/TCP + REALITY, VLESS + XHTTP + TLS и VLESS + XHTTP + REALITY;
- добавлены XHTTP Host, Path и Mode;
- добавлены TLS ALPN и Allow insecure с безопасным значением по умолчанию;
- добавлена миграция существующей SQLite-базы без потери outbounds;
- системные `direct` и `blocked` явно показаны в начале страницы;
- добавлены пояснения о текущих ограничениях и будущих протоколах.

## 0.9.4 — Clean reinstall and EC2 micro fixes

- добавлен `deploy/uninstall.sh` для чистой повторной установки;
- загрузчик из GitHub больше не зависит от Release и восстанавливает права `.sh`;
- email для Let's Encrypt больше не запрашивается;
- уточнены вопросы о доменах Xray и панели;
- для серверов с малым объёмом RAM автоматически создаётся swap 2 ГиБ;
- Waitress использует 2 потока, обслуживание запускается каждые 15 минут;
- статистика Xray по умолчанию выключена;
- QR-библиотека загружается только при открытии страницы ссылки;
- pip работает без локального кэша.

## 0.9.3 — EC2 Ready

- добавлен мастер первой установки `deploy/ec2-first-install.sh`;
- мастер устанавливает Xray 26.3.27, SG-Panel, Nginx и Certbot;
- добавлена проверка A-записей Dynu перед выпуском сертификата;
- HTTPS-порт панели по умолчанию изменён на `61443`;
- внешний порт панели выбирается пользователем в диапазоне `49152–65535`;
- выбранный порт резервируется через `net.ipv4.ip_local_reserved_ports`;
- backend панели на EC2 слушает только `127.0.0.1:8080`;
- порт `8080` не требуется открывать в AWS Security Group;
- Nginx слушает порт `80` для ACME HTTP-01 и выбранный HTTPS-порт панели;
- добавлен deploy hook для перезагрузки Nginx после обновления сертификата;
- `configure-https.sh` теперь использует порт `61443` по умолчанию;
- `install-gui.sh` поддерживает переменные окружения для bind address и порта;
- сохранены все функции и дизайн версии 0.9.2.

## 0.9.2 — Compact Actions

- компактные кнопки по ширине текста;
- действия форм выровнены и сгруппированы;
- сохранена мобильная адаптация.
