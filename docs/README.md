## Как читать названия

В интерфейсе технические разделы оставлены на английском:

- **Inbound** — как трафик входит в Xray;
- **Outbounds** — какие выходы доступны Xray;
- **Routing** — какой трафик направлять в каждый выход;
- **Xray Config** — итоговый JSON и расширенное редактирование;
- **Diagnostics** — проверка служб, сети и журналов.

Остальные названия интерфейса и инструкции даны на русском.
